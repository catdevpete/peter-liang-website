Super Pixel Smash

Project Description

Super Pixel Smash is a Virtual Reality game developed on the Unity engine.

The game was commissioned by Mako Labs, with the intent to create a 1-player tennis game with retro-like graphics.

The game was in development during late 2016, and published by Mako Labs in early 2017.

Project Details

  • Made in Unity
  • Developed for the HTC Vive
  • Available on the Steam store
